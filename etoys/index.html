This is a very early version of SqueakJS running an Etoys image. Feedback and contributions welcome!

SqueakJS is a Squeak VM written in pure JavaScript by Bert Freudenberg. It is normally developed and used in the Lively environment. This page demonstrates that SqueakJS also works outside Lively (and e.g. on an iPad). For the full interface, please visit the Lively SqueakJS page.
